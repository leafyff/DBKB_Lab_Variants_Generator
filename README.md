# DBKB_Lab_Variants_Generator
A student who has completed laboratory work must defend the theoretical part. The program selects random question numbers, which the teacher assigns to the student.

## How to lauch

#### Method 1
- Download .exe file from the "releases"
- Lauch .exe file

#### Method 2
- Clone repository
- Lauch main.py with your IDE or with CMD
- Insure you have python version 3.12+ and PyQt6. Otherwise download them
